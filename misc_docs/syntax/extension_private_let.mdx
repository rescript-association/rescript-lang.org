`%%private` is used to make let bindings private.

<CodeTab labels={["ReScript", "JS Output"]}>

```res
module Calc = {
  %%private(let mult = (x, y) => x * y)

  let double = x => mult(x, 2)
  let triple = x => mult(x, 3)
}
```

```js
function $$double(x) {
  return x << 1;
}

function triple(x) {
  return Math.imul(x, 3);
}

var Calc = {
  $$double: $$double,
  triple: triple,
};
```

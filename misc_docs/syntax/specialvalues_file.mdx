`__FILE__` returns the name of the file currently being parsed by the compiler.

<CodeTab labels={["ReScript", "JS Output"]}>

```res example
// testmodule.res
__FILE__->Js.log
```

```js
console.log("testmodule.res");
```

</CodeTab>

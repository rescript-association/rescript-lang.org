`%re` is used to create JavaScript regular expressions.

<CodeTab labels={["ReScript", "JS Output"]}>

```res
let regex = %re("/^hello/")
let result = regex->Js.Re.test_("hello world")
```

```js
var regex = /^hello/;
var result = regex.test("hello world");
```

</CodeTab>

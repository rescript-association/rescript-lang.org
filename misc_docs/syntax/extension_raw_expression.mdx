`%raw` is used to embed "raw JavaScript expressions".

<CodeTab labels={["ReScript", "JS Output"]}>

```res
let canUseCanvas: unit => bool = %raw(`
  function canUseCanvas() {
    return !!document.createElement('canvas').getContext;
  }
`)
```

```js
var canUseCanvas = function canUseCanvas() {
  return !!document.createElement("canvas").getContext;
};
```

</CodeTab>

See [`%%raw`](#raw-top-level-expression) for embedding top level blocks of JavaScript code rather than expressions.

`__LOC__` returns a string of the location in the file at which you put this very expression.
The string format is `"File %S, line %d, characters %d-%d"`

<CodeTab labels={["ReScript", "JS Output"]}>

```res example
__LOC__->Js.log
```

```js
console.log("File \"testmodule.res\", line 1, characters 0-7");
```

</CodeTab>

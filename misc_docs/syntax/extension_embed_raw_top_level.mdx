`%%raw` is used to embed top level JavaScript code.

<CodeTab labels={["ReScript", "JS Output"]}>

```res
%%raw(`
  const message = "hello";

  function greet(m) {
    console.log(m)
  }

  greet(message)
`)
```

```js
const message = "hello";

function greet(m) {
  console.log(m);
}

greet(message);
```

</CodeTab>

See `%raw` for embedding JavaScript expressions rather than top level blocks of code.
